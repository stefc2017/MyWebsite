Programmer Dev Tools Sql

The users of my app can practice their sql writing by choosing to insert, remove or search for data in the database. There is even instructions on how to make sql statements.

Programmer Dev Tools Modal

When the user clicks on the question mark beside the action to perform on the database, a modal pops up with instructions on how to create a sql statement for that action.

Programmer Dev Tools Insert

This page displays the table name and the columns in the database along with their data type so that the user can make sql queries easily and learn how to do so for the future

Programmer Dev Tools Search

The user can search for data and filter it by a certain column, ascending or descending order and even how many results to display.

Programmer Dev Tools Results

This page displays all the results based on the filters chosen on the previous page. The results can span many pages depending on how many entries are in the database and that functionality is supported.
