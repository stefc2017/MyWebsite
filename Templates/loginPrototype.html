LoginPrototype Home Page

This is a prototype for an app login. The user can register(or make an account) and then log in to the app. The code for this app can be used for other apps that need a user to have an account to access the app.

This app has a lot of business-logic for doing error checking to make sure the fields are filled and the information matches the records in the database.

LoginPrototype Register Page

The business-logic will make a call to the database to make sure there is no previous entry in the database with the same username to avoid duplicate entries.

LoginPrototype Register Page

On submit the business-logic will make sure every field matches what is supposed to be entered and will make sure the password and re-entered password matches.

LoginPrototype Login

A call will be made to make sure the username entered is found in the database and that the password entered matches the password in the database for that username.

LoginPrototype Logged In

After the credentials are verified, the user gets in the app and then this is where the content for that user would appear. The back button is disabled on the phone so the user can't log out and come back in.
